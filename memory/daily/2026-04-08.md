## 教訓：Puppeteer 中文字型

- Docker Alpine 容器內無中文字型 → 日期選擇器中文按鈕渲染成方塊 → 操作失敗
- 解法：`apk add font-noto-cjk`（Alpine）或 `apt install fonts-noto-cjk`（Debian）
- VPS host 已裝 `fonts-noto-cjk`，回報機器人不受影響
- CS Bot 容器（node:20-alpine）未來要跑 Puppeteer 必須加字型
