### mini888ph.xyz 開通流程（已完成）

2. GoDaddy API 改 NS → `irena.ns.cloudflare.com` + `owen.ns.cloudflare.com`
3. CF API 建 zone + 加 DNS：
   - A `@` → `67.209.125.243` (proxied 橘雲)
   - CNAME `www` → `mini888ph.xyz` (proxied)

5. 關鍵：wp-config.php 注入動態 site URL，讓 WP 不強制 redirect 回 mini88bet.com
   ```php
   if (isset($_SERVER['HTTP_HOST'])) {

   ```

### 經驗教訓 / 標準操作流程（SOP）
- **ISP 封鎖博弈域名的典型判斷**：
  - DNS 能正常解析（返回 CF IP） → 不是 DNS 污染
  - 直連 IP 能通但打域名不通 → SNI/DPI 封鎖
  - 只能換新域名（+ CF Proxy）繞過
- **便宜跳板域名**：`.xyz / .online / .site` 約 $0.99~$3/年
- **WordPress 多域名共用技巧**：用 `$_SERVER['HTTP_HOST']` 動態定義 WP_HOME/WP_SITEURL
- **GoDaddy API 購買限制**：需要帳號設好預設信用卡 + 允許 API 購買；否則讓用戶手動買比較快
