## 數據抓取重點

### 即時監控數據
1. **財務健康度**
   - 充提差（充值 - 提現）
   - 毛利（投注 - 中獎）
   - 平台資金餘額

2. **玩家活躍度**
   - 登入人數
   - 投注人數/金額
   - 充值人數/金額

3. **遊戲表現**
   - 熱門遊戲投注額
   - 遊戲盈虧排名

4. **風險指標**
   - 大額玩家損益
   - 異常投注模式
   - 充值成功率

### 報告頻率建議
- **每小時**：核心指標（充值、提現、投注、在線人數）
- **每日**：詳細報表（玩家排名、遊戲排名、財務摘要）
- **每週/月**：趨勢分析、風險報告

## 技術備註
- 後台使用動態加載（SPA），URL 可能不變
- 需要處理登入狀態維持（cookie/session）
- 數據以表格形式呈現，容易解析
- 有時間區選擇器（GMT+8）
