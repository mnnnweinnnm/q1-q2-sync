## 數據抓取優先順序

### 1. 核心指標 (每小時) -> 來源：首頁 (運營概括)
- 充值、提現、投注、中獎、註冊人數。

### 2. 損益報表 (每日) -> 來源：報表管理 -> 平台損益總表
- 詳細列出各產品損益、平台總營收。

### 3. 風險監控 -> 來源：玩家資金/流水返水
- 異常大額提現、異常連勝紀錄。
