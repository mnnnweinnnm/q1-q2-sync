### 📌 LiveChat 圖片發送流程（重要教訓，永久記錄）

**LiveChat API 不支援外部圖片 URL！** 必須用以下流程：

```
1. axios.get(imageUrl, { responseType: 'arraybuffer' })  // 下載圖片到 Buffer
2. POST https://api.livechatinc.com/v3.6/agent/action/upload_file  // 上傳到 LiveChat CDN
   - Content-Type: multipart/form-data
   - Authorization: Bearer <bot_token>
   - Body: form-data with file field
   - 回傳: { url: "https://cdn.livechat-files.com/..." }
3. send_event({ type: 'file', url: cdnUrl })  // 用 CDN URL 發送 file event
4. sendMessage(chatId, caption)  // 文字說明另外發
```

**踩過的坑：**
- ❌ `message` event + `attachments` → 被靜默忽略，不會報錯
- ❌ `file` event + 外部 URL → 不顯示
- ✅ `file` event + LiveChat CDN URL → 正確顯示
