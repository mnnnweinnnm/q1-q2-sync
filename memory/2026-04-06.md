## AutoSSL vs Cloudflare Proxy 衝突
**根因**：Let's Encrypt 要讀 `/.well-known/acme-challenge/` 驗證檔，CF 可能快取或擋到。
**解決方案**：
- 最省事：CF SSL Mode = Flexible / Full（非 strict），不用管 AutoSSL
- 最安全：CF Origin Certificate（15 年）裝 cPanel + Full (strict) ← 我們選的
- 折衷：加 Page Rule bypass cache on `/.well-known/acme-challenge/*`
