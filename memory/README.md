# 記憶系統架構

## 資料夾結構
```
memory/
├── daily/           # 每日原始記錄（按日期）
├── topics/          # 主題整合檔案
├── projects/        # 專案相關記錄
├── archive/         # 封存舊檔案
└── README.md        # 本檔案
```

## 維護機制
- **每週整理**：將日誌內容分類整合到主題檔案
- **每月壓縮**：重要內容寫入 MEMORY.md，舊檔案移至 archive/
- **定期清理**：移除過時或重複資訊

## 搜尋方式
- 語義搜尋（需設定 OpenAI API key）
- 關鍵字搜尋
- 檔案分類瀏覽

## 記錄原則
- 重要決策和變更
- 系統設定和問題解決
- 專案進度和成果
- 學習心得和教訓
